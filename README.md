## 📖 功能特性

- 🎨 **自定義畫布引擎**：從零開發基於 Canvas API 的引擎，支援高度客製化的物聯網控制佈局。
- ⚡ **即時 MQTT 通訊**：優化數據交換機制，確保與設備連線的穩定度與響應速度。
- 💡 **創意實踐與自學**：展現強大的自學能力，將創新的 UI 想法與功能需求轉化為功能完備的產品。
- 🔌 **軟硬體整合**：透過 MQTT 協議輕鬆串接 Arduino 等硬體終端，實現完整的控制鏈結。
